## Flutter Nylo Template ##

Combined with Nylo and my favourite libraries

## Requirements ##

- Flutter >= 2.5.0
- Android Studio 
- Xcode
- VsCode

## Getting Started ##

`flutter run`

## Core libraries ##

- Nylo Framework >= 2.2.6
- Injectable (dependency injection)
- retrofit (annotation api client)
- flutter_bloc (state management)
